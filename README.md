## 🛠️ Tech Stack

- **Core:** React.js + Vite
- **Styling:** CSS Modules (Vanilla CSS)
- **Effects:** Decipher Text animations, Hologram UI, Starfield backgrounds, Glassmorphism
- **Routing:** React Router DOM
- **Deployment:** GitHub Pages

## 📂 Project Manifest

### [MANIFEST_NODE_01] Military Data Transfer System

- **Tech:** Python, Steganography, Security
- A secure data transfer system implementing block rotation algorithms for tamper-resistant communication.

### [MANIFEST_NODE_02] Shop Your Hood

- **Tech:** Python, Django
- An offline shopping management system supporting local vendors with product listing and order management.

### [MANIFEST_NODE_03] EV Charging Station System

- **Tech:** Python, Django
- A smart alert-based system for detecting nearby electric vehicles and notifying users of available charging stations.

## ⚙️ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/ayishashadhi/aysha-portfolio.git
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run locally:**

   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🚢 Deployment

This project is configured to deploy automatically to GitHub Pages using the `gh-pages` package. To deploy new changes, run:

```bash
npm run deploy
```
